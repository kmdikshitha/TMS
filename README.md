# 🌍 Tour Management System (TMS)

An end-to-end platform to streamline travel planning—designed with modern architecture, scalable services, and secure authentication. The system enables users to explore destinations, plan itineraries, and manage bookings—all through an intuitive interface.

## ✨ Core Features

### 🔍 Search & Explore
- Discover travel destinations  
- Accommodation and transport search

### 🔐 Authentication & Authorization
- Login/Signup with credentials  
- OAuth via Google (Facebook planned)  
- Password recovery  
- Role-based protected routes (e.g., booking, itinerary)

### 🗺️ Itinerary Management
- Create and manage travel plans  
- Day-by-day trip planning  
- Fully customizable itineraries

## 🧩 Architecture Principles

- **Decoupled Services:** Frontend ↔ Backend via REST APIs  
- **Reusable Business Logic:** Centralized in Spring Boot backend  
- **Secure Auth Layer:** OAuth microservice for Google/Facebook login  
- **Extensible:** Microservice-ready structure for scaling and modular additions
